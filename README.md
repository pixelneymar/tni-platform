# TNI - Transaction Network Intelligence

A professional-grade data visualization interface featuring an interactive, zoomable bubble chart of the global economy.

## Features

- **195 Countries** with realistic GDP data
- - **20 Industries** per country (3,900 industry entries)
  - - **27,000+ Companies** with revenue data
    - - **Interactive Navigation**: Click to drill down from Global → Country → Industry → Companies
      - - **Search**: Find any country, industry, or company instantly
        - - **Professional Design**: Bloomberg Terminal-inspired dark theme
         
          - ## Quick Start
         
          - 1. Clone this repository
            2. 2. Open `src/index.html` in a web browser
               3. 3. Or visit the GitHub Pages site
                 
                  4. ## Technology
                 
                  5. - **D3.js v7** for force-directed visualization
                     - - **Vanilla JavaScript** - no framework dependencies
                       - - **CSS3** with custom properties for theming
